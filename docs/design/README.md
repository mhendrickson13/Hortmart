# VideoLecture UX/UI Spec (Mobile-first)

This folder contains a Figma-ready UX/UI specification aligned to the provided course-player mockup and extended to cover:

- Learner **course progress & completion** (mobile-first)
- Creator/Admin **course creation**, **lesson management**, **basic analytics**, **settings/profile**
- A **light, clean, premium** SaaS-style **design system**
